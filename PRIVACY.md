# Privacy And Sanitization Notes

This portfolio package is intentionally sanitized.

Do not include:

- real hostnames
- real IP addresses
- real running configs
- raw change logs
- raw Forward Networks payloads
- screenshots from private systems
- API keys, tokens, secrets, or credentials
- generated executables
- build folders or caches

Use only:

- placeholder hostnames
- documentation IP ranges such as `192.0.2.0/24`, `198.51.100.0/24`, and `203.0.113.0/24`
- short fake config snippets
- sanitized JSON fixtures
- generated example Markdown/SVG artifacts

Before publishing publicly, run a final review for:

- company names
- internal naming conventions
- real domains
- real subnets
- usernames
- serial numbers
- file paths that reveal private environments
- logs or command output copied from production systems
